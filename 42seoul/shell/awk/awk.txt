awk는 파일로부터 레코드를 선택하고, 선택된 레코드에 포함된 값을 조작하거나 데이터화하는 것을 목적으로 사용하는 프로그램이다. 즉, awk 명령의 입력으로 지정된 파일로부터 데이터를 분류한 다음, 분류된 텍스트 데이터를 바탕으로 패턴 매칭 여부를 검사하거나 데이터 조작 및 연산등의 액션을 수행하고, 그 결과를 출력하는 기능을 수행한다.

awk로 할 수 있는 것들
텍스트 파일의 전체 내용 출력.
파일의 특정 필드만 출력.
특정 필드에 문자열을 추가해서 출력.
패턴이 포함된 레코드 출력.
특정 필드에 연산 수행 결과 출력.
필드 값 비교에 따라 레코드 출력.


FILE이라는 파일이 있다고 가정
A 10 20 30
B 40 50 60
C 70 80 90
위의 내용으로 되어있다.

파일의 내용 출력
awk '{print}' FILE
A 10 20 30
B 40 50 60
C 70 80 90

파일의 첫 번째, 두 번째 필드만 출력
awk '{print $1, $2}' FILE
A 10
B 40
C 70

특정 필드에 문자열 추가하여 출력
awk '{print "user:"$1, "score:"$2}' FILE
user:A score:10
user:B score:40
user:C score:70

문자열 A를 포함하는 레코드만 출력
awk '/A/' FILE

세 번째 필드의 합 계산
awk '(sum += $3) END {print sum}' FILE

두 번째 필드가 40이면 출력
awk '$2 == 40 {print $0}' FILE

$0은 모든 필드를 의미한다..
$1은 첫 번째 필드
문자열은 ""으로 나타낸다.


awk
입력 데이터로부터 주어진 패턴을 포함하는 라인을 찾기 위해 파일의 내용을 탐색한 다음, 패턴에 일치하는 라인이 발견되면 해당 라인에 대해 지정된 액션을 실행한다. 이 과정을 입력 파일의 끝을 만날 때까지 수행한다.

awk는 기본적으로 입력 데이터를 라인 단위의 레코드로 인식한다.
