다수의 데이터를 저장하고 처리하는 경우에 유용하게 사용할 수 있는 것이 배열이다. 

int arr1[7];
float arr2[10];
double arr3[12];


배열의 길이정보는 상수로 할 것.

배열의 위치 정보를 명시하는 인덱스 값은 1이 아닌 0에서부터 시작한다.


배열 선언과 동시에 초기화

int arr1[5] = {1, 2, 3, 4, 5};

int arr2[] = {1, 2, 3, 4, 5, 6, 7}; // 컴파일러에 의해서 자동으로 7이 삽입된다.

int arr3[5] = {1, 2}; // 3, 4, 5번째 배열요소는 0으로 채워짐


char str[14] = "Good morning!"; // 문자열은 마지막에 \0가 들어간다.
위의 문자열의 길이는 13이지만 실제 문자열의 길이는 14가된다. 모든 문자열의 끝에는 \0이라는 특수문자가 자동으로 삽입이 되기 때문이다.
따라서 문자열의 저장을 목적으로 char형 배열을 선언할 경우에는 특수문자 \0이 저장될 공간까지 고려해서 배열의 길이를 결정해야 한다. \0를 널 문자라고 한다.

널 문자의 아스키 코드 값은 0이다. 이를 문자의 형태로 출력할 경우 아무런 출력이 발생하지 않는다. 공백 문자의 아스키 코드값은 32로 널문자와는 다르다.


scanf함수는 공백을 기준으로 문자열을 나눈다. 따라서 문장을 입력받기에는 적합하지 않다.
