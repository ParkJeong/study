포인터의 포인터는 포인터 변수를 가리키는 또 다른 포인터 변수를 뜻한다.
이중 포인터 또는 더블 포인터라 부른다.
int ** dptr;	// int형 이중 포인터

int * ptr;		// 싱글 포인터 변수
int ** dptr;	// 더블 포인터 변수

포인터 변수는 종류에 상관없이 무조건 주소 값을 저장하는 변수이다. 다만 차이가 나는 것은 포인터 변수가 가리키는 대상일 뿐이다.


포인터 변수의 선언에 있어서 * 연산자가 둘 이상 사용되어 선언되는 포인터 변수를 가리켜 다중 포인터 변수라고 한다. 포인터 변수 선언시 * 연산자는 얼마든지 추가될 수 있다.


포인터가 필요한 이유
함수 내에서 함수 외부에 선언된 변수에 접근하는 방법을 제시해준다.
